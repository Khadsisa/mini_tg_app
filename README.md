Мини-интеграция с Telegram
(принимает текст из файла .txt и отправляет этот текст в выбранный приватный Telegram-чат через Telegram-бота)

Для создания:
1. Установите: pip install requests
2. Создайте переменные- токен бота в config.py, айди чата и путь к файлу:
   FROM CONFIG IMPORT TOKEN;
   CHAT_ID, FILE_PATH
3. Далее следует создание функции по отправке текста при помощи response
4. В функции main объедините чтение файла txt с отправкой в чат, при помощи ранее созданной функции
5. Запустите приложение
